# Support

## Community Support

This is a **community-driven open-source project** without guaranteed SLA or response times.

### Where to Ask Questions

- **GitHub Issues:** [Open an issue](https://github.com/ranas-mukminov/Zabbix-templates-and-scripts/issues) for:
  - Bug reports
  - Feature requests
  - Template compatibility questions
  - Documentation improvements
  - General questions about using the templates

- **GitHub Discussions:** For broader questions and community interaction (if enabled)

- **Zabbix Community Forums:** For general Zabbix questions (not specific to this collection):
  - [https://www.zabbix.com/forum/](https://www.zabbix.com/forum/)

### Response Time

- **No guaranteed response time** ‚Äî this is a volunteer-maintained project
- Issues will be addressed on a best-effort basis
- Community members are encouraged to help each other

### Before Opening an Issue

Please:
1. **Search existing issues** ‚Äî your question may already be answered
2. **Check the README** ‚Äî both [README.md](README.md) and [README.ru.md](README.ru.md)
3. **Review template-specific documentation** ‚Äî most templates have their own README files
4. **Test in the Docker demo environment** ‚Äî see [examples/README.md](examples/README.md)
